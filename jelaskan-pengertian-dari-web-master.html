18. 1. Jelaskan apa tugas profesi dalam pengembangan aplikasi web berikut : a. Web Designer b. Web Programmer c. Web Administrator d. Web Master


Kelas: IX

Mata Pelajaran: Teknologi Informasi

Materi:  World Wide Web        

Kata Kunci:  web, http, internet

 

Tugas profesi dalam pengembangan aplikasi web berikut:


a. Web Designer: merancang susunan dan tampilan suatu situs pemrograman web


b. Web Programmer: membuat sistem dan halaman web sesuai dengan rancangan, dengan menggunakan bahasa pemrograman web


c. Web Administrator: menjaga kelangsungan kerja sistem web sehari-hari


d. Web Master: tugasnya adalah gabungan dari ketiga tugas di atas

 

Jawaban panjang:

 

a. Web Designer:

 

Ketika suatu web akan dibuat, maka disusun terlebih dahulu rancangan bentuk halaman, serta susunan alur halaman dalam bentuk link dari satu halaman ke halaman lain. Jika web menggunakan sistem halaman dinamik, maka database yang digunakan juga perlu dirancang

Merancang susunan dan tampilan suatu situs pemrograman web ini adalah tugas dari web designer.


b. Web Programmer:

 

Ketika sudah ada rancangan bentuk halaman dan alur situs, maka yang membuat sistem dan halaman web sesuai dengan rancangan, dengan menggunakan bahasa pemrograman web adalah Web Programmer. Bahasa pemrograman yang digunakan adalah HTML dengan dapat ditambah dengan Ruby, PHP atau ASP untuk logika halaman, CSS untuk menata tampilan halaman web, SQL untuk menghubungkan halaman web dengan database serta JavaScript untuk tampilan interaktif.


c. Web Administrator:

 

Ketika halaman web sudah jadi maka yang menjaga kelangsungan kerja sistem web sehari-hari adalah Web Administrator. Halaman web memerlukan orang yang rutin mengecek jalanya web apakah ada permasalahan atau tidak.


d. Web Master:

 

Web master tugasnya adalah gabungan dari ketiga tugas di atas. Web master harus memiliki skill yang dibutuhkan untuk melakukan desain, pembuatan dan perawatan situs web.

1. Pengertian Web server dan fungsinya

Web Server adalah perangkat lunak yang berfungsi sebagai penerima permintaan yang dikirimkan melalui browser kemudian memberikan tanggapan permintaan dalam bentuk situs web atau lebih umumnya dalam dokumen HTML.

Namun,Web Server dapat mempunya dua pengertian berbeda yaitu sebagai bagian dari perangkat keras Hardware dan peranglat lunak Software (Penjelasannya ada di atas yang Aku jawab).

Fungsi dari Web server adalah :

Memastikan semua modul yang dibutuhkan tersedia dan siap digunakan .Membersihkan penyimpanan,cache,dan module yang tidak terpakai.Melakukan pemeriksaan keamanan terhadap http request yang dikirimkan browser.

Bernama CERN httpd yang diusulkan oleh Sir Tim Berners-Lee. Web Server ini berjalan pada server NeXT.

NeXT merupakan perusahaan yang didirikan oleh Steve Jobs setelah keluar dari Apple.

Pada Tahun yang sama ditemukan Browser pertama kalinya yang dinamakan WordWibeWeb.

Dalam hal ini, sekumpulan web dapat disebut dengan website. Contoh homepage seperti pada www.brainly.co.id.

Pembahasan

Web dan homepage merupakan bagian dari layanan internet yang berhubungan dengan informasi. Selain homepage dan web, dikenal pula istilah webpage dan website. Keseluruhan hal tersebut dapat diakses melalui bantuan peramban atau browser. Perbedaan homepage dengan webpage sendiri jika dianalogikan adalah cover dan isinya. Homepage lebih mengrah halaman depan dan webpage merupakan bagian-bagiannya.

Contoh : dalam Brainly, homepagenya adalah www.brainly.co.id yang berisi tampilan utama dan menu pilihan mapel, notifikasi, profil dan leaderboard sedangkan webpagenya dapat berupa www.brainly.co.id/profil/username-id  maupun www.brainly.co.id/tugas/idtugas  nantinya webpage ini akan digabungkan dan terkait sehingga disebut dengan website

Sedangkan, sekumpulan web dapat dikatakan sebagai website yang menggunakan www sebagai awalan dan diakhiri dengan domain seperti .com .org .co.id
