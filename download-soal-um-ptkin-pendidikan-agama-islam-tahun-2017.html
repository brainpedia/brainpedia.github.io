Rata-rata :

(4 + a + b + 6 + c + d + 8)/7 = 6

(18 + a + b + c + d) = 42

a + b + c + d = 24

Misal, a + b = x, maka c + d = 24 - x

Kombinasi a + b yang mungkin :

a + b = 4 + 4 = 8

a + b = 4 + 5 = 9

a + b = 4 + 6 = 10

a + b = 5 + 5 = 10

a + b = 5 + 6 = 11

a + b = 6 + 6 = 12

Kombinasi c + d yang mungkin :

c + d = 6 + 6 = 12

c + d = 6 + 7 = 13

c + d = 6 + 8 = 14

c + d = 7 + 7 = 14

c + d = 7 + 8 = 15

c + d = 8 + 8 = 16

Kombinasi yang mungkin :

(8, 16), (9, 15), (10, 14), (11, 13), (12, 12)

Semua kombinasi berjumlah satu, kecuali (10, 14) yang bisa dibentuk dari :

(4 + 6) + (6 + 8), (4 + 6) + (7 + 7), (5 + 5) + (6 + 8), (5 + 5) + (7 + 7), berjumlah empat buah.

Artinya :

(8, 16) = 1 susunan

(9, 15) = 1 susunan

(10, 14) = 4 susunan

(11, 13) = 1 susunan

(12, 12) = 1 susunan

Jadi, banyaknya susunan : 8

96. A

located => past participle; seharusnya noun juga karena ada and

Yang tepat adalah location


97. A

Yang tepat adalah comes from = berasal dari


98. A
